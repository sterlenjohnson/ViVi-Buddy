# Apple Silicon Specifications (2025)

*Last Updated: November 25, 2025*

Apple Silicon's Unified Memory Architecture (UMA) allows the CPU and GPU to share a massive pool of high-bandwidth memory, making Mac Studios and MacBook Pros excellent for running very large models (70B+) that don't fit on consumer GPUs.

## M5 Series (Late 2025)

The M5 chip debuted in late 2025, built on TSMC's enhanced 3nm process.

| Chip | Max Memory | Memory Bandwidth | CPU Cores | GPU Cores | Notes |
|------|------------|------------------|-----------|-----------|-------|
| **M5 (Base)** | 32 GB | ~120 GB/s | 10 | 10 | ~12% faster CPU, 36% faster GPU vs M4. |

## M4 Series (2024-2025)

| Chip | Max Memory | Memory Bandwidth | CPU Cores | GPU Cores | Notes |
|------|------------|------------------|-----------|-----------|-------|
| **M4 Max** | 128 GB | 546 GB/s | 16 | 40 | Excellent inference machine. |
| **M4 Pro** | 64 GB | 273 GB/s | 14 | 20 | Good mid-range option. |

## M3 Series (2023-2024)

| Chip | Max Memory | Memory Bandwidth | Notes |
|------|------------|------------------|-------|
| **M3 Ultra** | 192 GB | 800 GB/s | **Current Ultra Flagship.** |
| **M3 Max** | 128 GB | 300-400 GB/s | First to support 128GB in a laptop form factor. |

## M2 Series (2022-2023)

| Chip | Max Memory | Memory Bandwidth | Notes |
|------|------------|------------------|-------|
| **M2 Ultra** | 192 GB | 800 GB/s | **Capacity King.** The only consumer way to run 180B+ models locally. |
| **M2 Max** | 96 GB | 400 GB/s | Great value used option. |
